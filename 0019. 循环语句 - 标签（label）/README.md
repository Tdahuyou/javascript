## 5. 🤔 问：如何看待“标签”？

- A：通常在项目中，都不会使用标签。
- JS 允许在语句前面使用标签，这可以与 `break` 和 `continue` 结合使用，以便在复杂的循环结构中更灵活地控制流程。但由于标签和跳转语句大概率会 **降低代码的可读性**，建议在使用时保持谨慎，并优先考虑其他更直观的控制流程方式。
